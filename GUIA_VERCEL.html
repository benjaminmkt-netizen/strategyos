☁️ Guía de despliegue en Vercel

StrategyOS — Online en 15 minutos

Sin instalar nada en tu computadora. Todo desde el navegador.

1
Crea una cuenta en GitHub
GitHub es donde vive el código. Vercel lo lee desde ahí y publica la app automáticamente.
Ir a
→ github.com/signup · Crea tu cuenta gratis (con tu correo personal, no el del trabajo)
Si ya tienes cuenta
Salta directo al paso 2 ✓
2
Sube los archivos a GitHub
Crea un repositorio nuevo y sube los 3 archivos del ZIP que descargaste.
Archivos a subir (del ZIP strategyos-vercel)
📄index.htmlLa app completa
⚙️vercel.jsonConfig de Vercel
📁api/chat.jsEl proxy seguro
Pasos en GitHub
1
Entra a github.com/new
2
Nombre del repo: strategyos · Marca Public · Clic en Create repository
3
En la página del repo vacío, clic en "uploading an existing file"
4
Arrastra los 3 archivos (respetando que chat.js va dentro de una carpeta api)
5
Clic en Commit changes
⚠️ Importante — carpeta api
El archivo chat.js debe estar dentro de una carpeta llamada exactamente api. Al subir archivos en GitHub, puedes escribir api/chat.js como nombre del archivo y GitHub crea la carpeta solo.
3
Conecta Vercel con GitHub
Vercel leerá tu repositorio y publicará la app automáticamente. Cada vez que actualices el código en GitHub, Vercel se actualiza solo.
Ir a
→ vercel.com/signup · Elige "Continue with GitHub" (así quedan conectados automáticamente)
Importar el proyecto
1
En el dashboard de Vercel clic en Add New → Project
2
Vercel mostrará tus repos de GitHub. Busca strategyos y clic en Import
3
En la pantalla de configuración, deja todo como está. No cambies nada aún.
4
Todavía NO hagas clic en Deploy — primero agrega la API Key en el paso 4
4
Agrega tu API Key de Anthropic
Esta es la parte clave: la API key vive en el servidor de Vercel, no en el código. Nadie más puede verla.
Obtener tu API Key
Ve a console.anthropic.com/settings/keys → Create Key → cópiala (empieza con sk-ant-)
En la pantalla de configuración de Vercel
1
Expande la sección "Environment Variables"
2
En el campo Name escribe exactamente: ANTHROPIC_API_KEY
3
En el campo Value pega tu API key
4
Clic en Add
ANTHROPIC_API_KEY = sk-ant-api03-xxxxxxxxxxxxxxxx
✓ Ahora sí
Haz clic en Deploy. Vercel tardará unos 30 segundos en publicar la app.
5
¡Listo! Comparte la URL con tu equipo
Vercel te da una URL pública gratuita que funciona en cualquier dispositivo.
Tu URL quedará algo así
🌐 strategyos-tuusuario.vercel.app


Compártela con tu equipo por WhatsApp, Slack o email. Entra en Chrome, Safari, Edge — lo que sea.
Dominio personalizado (opcional, también gratis)
En Vercel → tu proyecto → Settings → Domains puedes poner un nombre personalizado como estrategia.tuempresa.com si tu empresa tiene dominio propio.

StrategyOS online ✦

Tu agente de marketing con IA disponible para todo tu equipo, desde cualquier lugar, sin instalar nada.
